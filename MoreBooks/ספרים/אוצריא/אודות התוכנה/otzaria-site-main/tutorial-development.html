יצירת "מזלג" (Fork) למאגר

"מזלג" הוא למעשה יצירת עותק אישי של המאגר המקורי אל חשבון ה-GitHub שלכם.

  • גשו ל-מאגר 'אוצריא'
  • לחצו על כפתור ה-Fork בפינה הימנית העליונה

הורדת (Clone) המאגר למחשב

כעת, נוריד את עותק המאגר האישי שלכם למחשב.

1

העתקת כתובת המאגר

בחשבון ה-GitHub שלכם, היכנסו למאגר המשוכפל, לחצו על כפתור ה-Code והעתיקו את כתובת ה-HTTPS.

2

אפשרות א': דרך עורך הקוד

  • פתחו את עורך הקוד

  • הקלידו: git clone
  • הדביקו את הכתובת של המאגר
  • בחרו clone from url
  • בחרו תיקיה (מומלץ תיקיה שאין בנתיב שלה עברית)

bug_report בדיקה ובנייה

הרצה במצב Debug

סיימתם? עכשיו צריך לבדוק שהכל תקין!

הקלידו בטרמינל של עורך הקוד:

flutter run --debug -d windows

זה יוריד אוטומטית בפעם הראשונה את ה-SDK הנצרך בשביל לדבג.

רענון שינויים

  • לחצו על r בטרמינל - השינויים ישתקפו אוטומטית (עובד בעיקר לשינויים בעיצוב)
  • לחצו על R (אות גדולה) - יפעיל את הדיבוג מחדש (לשינויים יותר כבדים)
  • לחצו על F5 - להפעלה עם ממשק גרפי פשוט

במצב זה הרענון הוא אוטומטי, ויש שם אייקון להפסקת הדיבוג או ריסטארט לדיבוג.
